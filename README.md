# Star Trek Fan Fiction Site

A community platform for Star Trek fans to share and discover fan-written stories set in the Star Trek universe.

## Features

- Browse and read Star Trek fan fiction stories
- Clean, responsive design inspired by LCARS interface
- Featured stories showcase
- Category browsing (TOS, TNG, DS9, Voyager, Enterprise, Discovery, etc.)

## Getting Started

Simply open `index.html` in your web browser to view the site.

For development with live reload:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
